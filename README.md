## Sobre o app

Aplicativo desenvolvido para treinamento do GetX em Flutter.
  - Criando Repositório;
  - Modelo (class);
  - Controllers;
  - View e pages;
  - Amplify

Os dados utilizados foram da API https://pokeapi.co/

Nela contém dados de pokemóns para simular uma pokedéx.
